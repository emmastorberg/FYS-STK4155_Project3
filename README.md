# FYS-STK4155 Project 3
## By Frida Lien and Emma Storberg

To install required libraries, run:
```
pip3 install -r requirements.txt
```

To run pde/main.py, run:
```
python3 -m pde.main
```

To run tests, run:
```
python3 -m pytest pde/tests.py
```
